Pusat Bantuan & Simulasi FAQ

Bagaimana cara kerja saldo simulasi?

Ini adalah simulasi offline (client-side). Anda bebas menambahkan saldo secara instan menggunakan menu "Top Up" tanpa transaksi nyata.

Apakah saya bisa menggunakan NIM asli saya?

Tentu! Jika NIM Anda tidak ada di daftar demo kami, silakan salin salah satu NIM demo yang sudah kami sediakan untuk menguji visual tabel cicilan SPP.

Bagaimana simulasi QRIS berjalan?

Saat memilih metode QRIS, sistem kami memanfaatkan library qrcode.js untuk generate QR code dinamis yang valid dan dilengkapi countdown timer simulasi 5 menit.

Kemana data transaksi disimpan?

Data Anda disimpan pada LocalStorage browser laptop/komputer Anda secara persistent. Anda bisa mengklik tombol 'Clear History' untuk menghapus data tersebut.
